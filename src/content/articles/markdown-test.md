# 長めの本文

これは長めの本文の表示確認です。  
Markdown記事では、見出しやコードだけでなく、普通の文章が長く続くこともあります。そのときに、行間、文字サイズ、段落間の余白が自然に見えるかが大切です。

淡い背景の上に本文を置く場合、文字色が濃すぎると重く見えます。一方で薄すぎると読みづらくなります。今のデザインでは、本文は落ち着いた黒寄りの色、アクセントは水色、補助的な要素にはラベンダーやピンクを使う方針です。
